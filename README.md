### Update appsettings.json

```json
{
  "Logging": {
    "LogLevel": {
      "Default": "Information",
      "Microsoft.AspNetCore": "Warning"
    }
  },
  "AllowedHosts": "*",
  "AI":{
    "OpenAI":{
      "ApiKey":"<Get the API Key from your OpenAi account>" 
    }
  }
}
```
